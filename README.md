# shizuoka-2019-pointcloud

[静岡県　富士山南東部・伊豆東部　点群データ](https://www.geospatial.jp/ckan/dataset/shizuoka-2019-pointcloud) のダウンロードを補助する非公式ツールです。
地図上で矩形選択した範囲の点群データの URL を取得することができます。

# [ALB データ](alb.html)

航空レーザ測深（ALB）で取得した3次元点群データのダウンロードページです。

# [LP データ](lp.html)

航空レーザ測量（LP）で取得した3次元点群データのダウンロードページです。

# [MMS データ](mms.html)

移動計測車両（MMS）で取得した3次元点群データのダウンロードページです。
